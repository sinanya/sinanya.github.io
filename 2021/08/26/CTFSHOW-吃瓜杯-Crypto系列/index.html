P是一个单位置换组矩阵，加密过程：简单的系数与矩阵相乘然后再三次方，同时发现n是可以分解的。脚本如下：

from Crypto.Util.number import *

p1 = 35200554341
p2 = 35592761021
p3 = 48105446183
p4 = 50240938337
p5 = 50769514811
p6 = 62183036099
p7 = 63345654131
p8 = 66104249843
p9 = 67515121673
p10 = 67704082961
p11 = 48479007809

n = 8870619487339789349033932217513908953609539651949986489986889710933094577873155191810742828503059670650154455297603719
e = 3
c = 6940158573485767169443582872275118843545217792197971962103010557916847970940437712181778807436191892307187137338300231
phin = (p1 - 1) * (p2 - 1) * (p3 - 1) * (p4 - 1) * (p5 - 1) * (p6 - 1) * (p7 - 1) * (p8 - 1) * (p9 - 1) * (p10 - 1) * (p11 - 1)
d = inverse(e, phin)
m = pow(c, d, n)
print(long_to_bytes(m))

运行脚本可得flag。

可知这是一题经典的转轮机密码，转轮机密码是一种多表替换密码。

工作原理：

​ 转轮密码机由多个转轮构成，每个转轮旋转的速度都不一样，比如有3个转轮，分别标号为1,2,3，其中1号转轮转动26个字母后，2号转轮就转动一个字母，当2号转轮转动26个字母后，3号转轮就转动1个字母。因此，当转轮密码机转动26x26x26次后，所有转轮恢复到初始状态，即3个转轮密码机的一个周期长度为26x26x26（17576）的多表代换密码。

from pycipher import Enigma
import string, itertools

for key in itertools.product(string.ascii_uppercase, repeat=3):
    cipher = Enigma(settings=('W', 'Y', 'F'), ringstellung=key, steckers=[('W', 'O'), ('D', 'E'), ('J', 'B'), ('H', 'N'), ('X', 'I')])
    m = cipher.decipher('MXKXBTIOOZHFTGGTTPTRNXJUGASUTVBNSNGS'.upper())
    if('CTFSHOW' in m):
        print(m.lower())
